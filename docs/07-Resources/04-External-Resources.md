## Getting Help

### Community Forums
1. Search for existing answers
2. Post detailed question if needed
3. Include workflow snippet if relevant
4. Tag appropriately

### Support
- Licensed users: Alteryx Support Portal
- Trial users: Community forums

### Tips for Good Questions
1. Describe what you're trying to do
2. Explain what you've tried
3. Include sample data
4. Share error messages
5. Post workflow screenshots
