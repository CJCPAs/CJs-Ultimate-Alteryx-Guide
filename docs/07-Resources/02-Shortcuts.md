# Keyboard Shortcuts

[Home](../../README.md) > [Resources](./README.md) > Shortcuts

---

Master these shortcuts to speed up your workflow building.

---

## File Operations

| Shortcut | Action |
|----------|--------|
| `Ctrl + N` | New workflow |
| `Ctrl + O` | Open workflow |
| `Ctrl + S` | Save workflow |
| `Ctrl + Shift + S` | Save As |
| `Ctrl + W` | Close workflow |

---

## Workflow Execution

| Shortcut | Action |
|----------|--------|
| `Ctrl + R` | Run workflow |
| `Escape` | Stop running workflow |
| `Ctrl + Shift + R` | Run with data profile |

---

## Edit Operations

| Shortcut | Action |
|----------|--------|
| `Ctrl + Z` | Undo |
| `Ctrl + Y` | Redo |
| `Ctrl + X` | Cut selected tools |
| `Ctrl + C` | Copy selected tools |
| `Ctrl + V` | Paste tools |
| `Delete` | Delete selected tools |
| `Ctrl + A` | Select all tools |
| `Ctrl + F` | Find in workflow |

## Canvas Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl + 0` | Fit workflow to window |
| `Ctrl + 1` | Actual size (100% zoom) |
| `Ctrl + Mouse Wheel` | Zoom in/out |
| `Space + Drag` | Pan canvas |
| `Middle Mouse + Drag` | Pan canvas |
| `Arrow Keys` | Move selected tools |

## Custom Favorites

Add frequently used tools to favorites:
1. Right-click any tool in palette
2. Select "Add to Favorites"
3. Access from Favorites category

---

## Pro Tips

### Quick Tool Search
1. Click in Tool Palette search box
2. Start typing tool name
3. Press Enter to add first match

### Duplicate Tools Quickly
`Ctrl + Drag` creates a copy connected to the same source.

### Align Tools
1. Select multiple tools
2. Right-click → Align
3. Choose alignment option

### Quick Browse
Right-click any connection → Add Browse Tool
