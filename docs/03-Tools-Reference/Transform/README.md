# Transform Tools

[Home](../../../README.md) > [Tools Reference](../README.md) > Transform

Tools for reshaping and aggregating data.

## Tools in This Category

| Tool | Purpose |
|------|---------|
| [Summarize](01-Summarize.md) | Group and aggregate data |
| [Running Total](02-Running-Total.md) | Calculate cumulative values |
| [Transpose](03-Transpose.md) | Convert columns to rows |
| [Cross Tab](04-Cross-Tab.md) | Convert rows to columns (pivot) |
| [Sample](05-Sample.md) | Take subset of rows |
| [Unique](06-Unique.md) | Remove duplicates |
| [Tile](07-Tile.md) | Assign records to groups/buckets |

## Common Transformations

| I want to... | Use |
|--------------|-----|
| Sum/Count/Average by group | [Summarize](01-Summarize.md) |
| Get running totals | [Running Total](02-Running-Total.md) |
| Pivot columns to rows | [Transpose](03-Transpose.md) |
| Pivot rows to columns | [Cross Tab](04-Cross-Tab.md) |
| Remove duplicate rows | [Unique](06-Unique.md) |
