## Quick Reference

### Reading Data
- CSV/TXT files → Input Data
- Excel files → Input Data
- Databases → Input Data or Connect In-DB
- APIs → Download tool (then parse)

### Writing Data
- Files → Output Data
- Databases → Output Data
- Multiple sheets → Output Data with sheet names
