## Most Used Tools

### The Essential 10

1. **Input Data** - Read files and databases
2. **Output Data** - Write results
3. **Select** - Choose and rename columns
4. **Filter** - Keep or exclude rows
5. **Formula** - Create calculations
6. **Join** - Combine data on keys
7. **Union** - Stack datasets
8. **Summarize** - Aggregate data
9. **Sort** - Order rows
10. **Browse** - Inspect data
