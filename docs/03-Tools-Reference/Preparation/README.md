Tools for cleaning, filtering, and transforming data.

## Tools in This Category

| Tool | Purpose |
|------|---------|
| [Select](01-Select.md) | Choose columns, rename, reorder, change types |
| [Filter](02-Filter.md) | Keep or exclude rows based on conditions |
| [Formula](03-Formula.md) | Create new columns, calculations |
| [Sort](04-Sort.md) | Order rows by column values |
| [Sample](05-Sample.md) | Take subset of rows |
| [Unique](06-Unique.md) | Remove duplicate rows |
| [Data Cleansing](07-Data-Cleansing.md) | Clean text, handle nulls |
| [Auto Field](08-Auto-Field.md) | Optimize field types |
| [Multi-Field Formula](09-Multi-Field-Formula.md) | Apply formula to multiple fields |
| [Multi-Row Formula](10-Multi-Row-Formula.md) | Reference other rows |
| [Record ID](11-Record-ID.md) | Add unique row numbers |
| [Imputation](12-Imputation.md) | Fill missing values |

## Most Commonly Used

1. **Select** - Every workflow needs field management
2. **Filter** - Keep only the data you need
3. **Formula** - The workhorse for calculations
4. **Sort** - Essential for ordering and grouping
