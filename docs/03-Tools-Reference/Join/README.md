# Join Tools

[Home](../../../README.md) > [Tools Reference](../README.md) > Join

---

Tools for combining data from multiple sources.

## Tools in This Category

| Tool | Purpose |
|------|---------|
| [Join](01-Join.md) | Combine two inputs on matching keys |
| [Union](02-Union.md) | Stack multiple inputs vertically |
| [Append Fields](03-Append-Fields.md) | Add columns from one source to another |
| [Find Replace](04-Find-Replace.md) | Replace values using lookup table |
| [Join Multiple](05-Join-Multiple.md) | Join more than two inputs |
| [Make Group](06-Make-Group.md) | Create hierarchical groups |

## Choosing the Right Tool

| I want to... | Use |
|--------------|-----|
| Match rows on a key field | [Join](01-Join.md) |
| Stack datasets with same columns | [Union](02-Union.md) |
| Add all columns from source B to A | [Append Fields](03-Append-Fields.md) |
| Replace values using a lookup | [Find Replace](04-Find-Replace.md) |
| Join 3+ sources at once | [Join Multiple](05-Join-Multiple.md) |

## Join Types Quick Reference

```
LEFT JOIN:   All from Left + matching from Right
INNER JOIN:  Only matching rows from both
FULL OUTER:  All from both, matching where possible
```
